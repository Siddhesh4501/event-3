CODE JUNKIE ++

The objective of the event is to test the knowledge and innovativeness of students in object-oriented programming. This event is for programmers having some experience in oop and will try to cover object-oriented aspects of c++, java and python programming languages. This event has been designed for people who love coding and who are passionate about programming.

Prize Money : Rs.25000/-

About Event

Round 1:
It will consist of multiple choice questions with single/multiple correct answer/s. Participating teams can take the test from only one of these three languages:
1. C++
2. Java
3. Python
For subsequent rounds, teams have to continue with their chosen language. The marking scheme will be announced on the spot. This round will be conducted online, but in our campus. You will be provided access to the test only if you report at our event venue during the allotted time slot.


Round 2:
The top teams from the Round 1 will qualify for this round. This round will be a rapid coding round where teams have to complete codes within the stipulated time. The output file needs to be submitted and will be judged on the basis of the time taken to code it. Further details will be informed at the time of event.


Round 3:
This round will be a proper coding round with relatively difficult questions. This time, teams will have to try to make the code both working as well as efficient. All possible inputs and corner cases shall be handled. Judging criteria will have great emphasis on execution time and memory usage of programs. More details will be disclosed on the spot.


TEAM AND FEE STRUCTURE:
Team Size : Max 2 participants per team.
Registration Fee : ₹ 50 per team.

RULES

1. For rounds 2 and 3, the environment – programming language, IDE, editor, compiler, interpreter, etc. will be anything you are comfortable with. But please note that:
a. OS will be GNU / Linux or Windows.
b. In round 2, some questions will be specifically related to C++, Java and Python. (You will be given an option to solve questions either of C++ or Java or Python).
c. For round 3, judging criteria will stress on program efficiency in terms of both time and space. Choose your programming language in this round wisely.

2. This event is open to all undergraduate students.

3. Participants need to bring their College ID cards and registration receipts at the time of reporting.

4. Participants are expected to give their correct contact details. (Email ID and phone number).

5. Rules may be changed without prior intimation. Participants are requested to check the MindSpark’21 website www.mind-spark.org regularly for updates

FAQS
